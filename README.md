# George

[![Docs](https://img.shields.io/badge/docs-gigurra.github.io%2Fgeorge-blue)](https://gigurra.github.io/george/)

A GitOps-driven LLM agent reconciliation engine.

> *Named after George Smiley, John le Carré's patient spymaster who manages networks of agents.*

## Documentation

**[View Documentation →](https://gigurra.github.io/george/)**

Documentation also lives in [`docs/`](docs/). To browse locally with mkdocs:

```bash
uv run --with mkdocs-material mkdocs serve
```

Then open http://localhost:8000
